# IT Helpdesk Ticket Management System | ServiceNow ITSM

## Overview
Developed a ServiceNow ITSM-based Helpdesk Ticket Management System to streamline enterprise support operations including incident handling, SLA tracking, user administration, workflow automation, and reporting. The project simulates real-world IT support processes followed in enterprise environments.

---

## Features
- Incident Management
- SLA Configuration & Tracking
- User & Group Administration
- Incident Lifecycle Management
- Reporting & Dashboard Creation
- Assignment Group Management
- Ticket Prioritization & Categorization
- Workflow Automation

---

## Technologies Used
- ServiceNow ITSM
- Incident Management
- SLA Management
- Reporting & Dashboards
- Flow Designer
- ServiceNow Tables & Forms

---

## Project Workflow
1. Created users, roles, and support assignment groups
2. Configured Incident Management workflows
3. Implemented SLA policies for ticket prioritization
4. Managed incident lifecycle from creation to resolution
5. Configured ticket categorization, assignment, and escalation
6. Generated dashboards and reports for ticket monitoring
7. Simulated enterprise support operations using ServiceNow modules

---

## Modules Implemented

### Incident Management
Configured incident forms, ticket creation, categorization, prioritization, assignment, and resolution workflows.

### SLA Management
Implemented SLA tracking based on incident priority with breach monitoring functionality.

### Reporting & Dashboards
Created incident reports and dashboards for operational monitoring and analytics.

### User Administration
Managed users, roles, and assignment groups for support operations.
