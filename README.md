# IT Helpdesk Ticket Management System | ServiceNow ITSM

## Overview
Developed a ServiceNow ITSM project for managing enterprise support incidents, SLA tracking, user administration, and reporting workflows.

## Features
- Incident Management
- SLA Configuration
- User & Group Administration
- Incident Lifecycle Tracking
- Reporting & Dashboard Creation

## Technologies Used
- ServiceNow ITSM
- Incident Management
- SLA Management
- Reporting & Dashboards

## Project Workflow
1. Created users and support groups
2. Configured incident management process
3. Implemented SLA tracking for hardware incidents
4. Managed ticket lifecycle from creation to resolution
5. Generated reports for incident monitoring
